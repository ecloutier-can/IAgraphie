# IAgraphie | Cégep de Thetford

Outil de génération de déclaration d'intégrité intellectuelle (IAgraphie) personnalisé pour le Cégep de Thetford.

## 🚀 Déploiement GitHub Pages

Ce projet est configuré pour se déployer automatiquement via **GitHub Actions**.

### Configuration requise (Action manuelle)

Pour que le déploiement fonctionne, vous devez activer les Pages sur votre dépôt GitHub :

1.  Allez dans **Settings** > **Pages**.
2.  Sous **Build and deployment** > **Source**, sélectionnez **GitHub Actions**.
3.  Une fois cette option choisie, le workflow `.github/workflows/deploy.yml` pourra publier le site à chaque mise à jour.

## 🛠 Dépendances

- Vite
- TypeScript
- FontAwesome 6
